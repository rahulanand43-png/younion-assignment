The Core PrincipleEvery source feeds the same Google Sheet → same HubSpot record → same score. Your SDR never needs to log into Bombora, G2, or Apollo. They only look at HubSpot. That's the entire goal of this integration.
01

Set Up Your Central Hub

Week 1 · Foundation · Free to start

Before connecting any source, you need the destination. This is the "one place" everything will flow into. Two tools: HubSpot CRM (the live operational system) and a Google Sheet (the weekly data aggregator and scoring engine).

1

Set Up HubSpot CRM with Intent Fields

Create the CRM structure that will receive all intent data from every source

Week 1 · Day 1–2 ✓ Free
  • Create HubSpot Free Account
    Go to hubspot.com → Start free → Sign up with Younion email. Free CRM gives you unlimited contacts, companies, and custom properties — everything you need for Phase 1.
    hubspot.com · Free forever
  • Add 6 Custom Properties to Company Records
    In HubSpot: Settings → Properties → Company Properties → Create property. Add these exact fields:
    HubSpot Custom Properties to Create
    // Go to: Settings → Properties → Company → Create property

Field 1:  Intent Score         → Type: Number
Field 2:  Intent Level         → Type: Dropdown (High / Medium / Low)
Field 3:  Intent Source        → Type: Multi-checkbox (Bombora, G2, Apollo, ZoomInfo, LinkedIn, Clutch)
Field 4:  Intent Topics        → Type: Text (e.g., "B2B Event Management, Brand Activation")
Field 5:  Intent Signal Date   → Type: Date (last signal received)
Field 6:  Surge Strength       → Type: Dropdown (High / Moderate / Low)
  • Create 3 List Views in HubSpot
    These become your daily working views. Go to Contacts → Companies → Create view:

    🔴 High Intent View

    • Filter: Intent Score ≥ 60
    • Sort: Intent Signal Date (newest first)
    • Columns: Company, Score, Source, Topics, Owner
    • This is SDR's daily starting point

    🟡 Medium Intent View

    • Filter: Intent Score 25–59
    • Sort: Score (high to low)
    • Columns: Company, Score, Source, Last Signal
    • Weekly SDR review list
  • Import Your Target Account List (TAL) into HubSpot
    Export your 200–500 TAL companies from LinkedIn or Google Sheets as CSV → HubSpot → Contacts → Import → Company. Map: Company Name, Domain, Industry, City. These records will be updated with intent scores as signals arrive.
    HubSpot Import · CSV upload
2

Build the Master Intent Google Sheet

Your weekly data aggregator — all 6 sources land here before flowing to HubSpot

Week 1 · Day 2–3 ✓ Free

This Google Sheet is your data warehouse before you have a real one. Create 8 tabs exactly as described below.

  • Create Google Sheet: "Younion Intent Master"
    Create a new Google Sheet with the following 8 tabs. Share with your sales and marketing team (View access for most, Edit for the person managing the data).
    Google Sheet Tab Structure
    Tab 1: MASTER_INTENT      ← Unified scored view (auto-calculated)
Tab 2: TAL                ← Your 200–500 target accounts (base data)
Tab 3: BOMBORA_RAW        ← Weekly paste from Bombora CSV export
Tab 4: G2_RAW             ← Weekly paste from G2 intent export
Tab 5: APOLLO_RAW         ← Weekly paste from Apollo export
Tab 6: ZOOMINFO_RAW       ← Weekly paste from ZoomInfo CSV
Tab 7: LINKEDIN_RAW       ← Weekly paste from LinkedIn analytics
Tab 8: CLUTCH_RAW         ← Weekly paste from Clutch enquiries
  • Set Up MASTER_INTENT Tab with Scoring Formulas
    This tab pulls from all 6 raw tabs and calculates a unified score. Each row = one company from your TAL.
    MASTER_INTENT Tab — Column Structure & Formulas
    Column A: Company Name        ← From TAL tab
Column B: Domain              ← From TAL tab
Column C: Industry            ← From TAL tab
Column D: Bombora Score       ← =IF(VLOOKUP(B2,BOMBORA_RAW!$A:$C,3,0)="High",35,IF(...=20,0))
Column E: G2 Score            ← =IF(VLOOKUP(B2,G2_RAW!$A:$B,2,0)="Compared",30,IF(...="Viewed",20,0))
Column F: Apollo Score        ← =IF(COUNTIF(APOLLO_RAW!$A:$A,B2)>0,15,0)
Column G: ZoomInfo Score      ← =IF(COUNTIF(ZOOMINFO_RAW!$A:$A,B2)>0,20,0)
Column H: LinkedIn Score      ← =IF(COUNTIF(LINKEDIN_RAW!$A:$A,B2)>0,15,0)
Column I: Clutch Score        ← =IF(COUNTIF(CLUTCH_RAW!$A:$A,B2)>0,10,0)
Column J: TOTAL SCORE         ← =SUM(D2:I2)
Column K: Intent Level        ← =IF(J2>=60,"HIGH",IF(J2>=25,"MEDIUM","LOW"))
Column L: Active Sources      ← Concatenation of which sources fired
Column M: Last Updated        ← =TODAY()
✅
What This AchievesEvery Monday morning, you paste 6 new CSV exports into their raw tabs. Column J recalculates every company's score automatically. Column K shows intent level. You now have one ranked list of every TAL company by intent — ready to push to HubSpot.

Connect Each Source

Weeks 2–4 · One source at a time

Connect each 3rd party source using the best available method — native integration where possible, Zapier automation where not, manual CSV export as the fallback. Start with the free ones, add paid later.

3

Connect G2 → HubSpot (Native Integration)

The easiest and highest-quality signal — buyers actively comparing vendors

Week 2 · Day 1 ✓ Free to start
  • Claim & Complete Younion's G2 Profile
    Go to g2.com → Search "Younion" or your services → Claim profile → Fill in: company description, services, case studies (minimum 3), pricing range, and upload logo. Ask 5 existing clients to leave reviews — this unlocks intent data.
    g2.com · Free profile
  • Enable G2 Buyer Intent Add-on
    In G2 admin → My Products → Buyer Intent → Enable. Free tier shows who viewed your profile. Paid tier (~$299/mo) shows who compared you vs competitors. Start with free, upgrade after 30 days if signals are valuable.
    G2 Intent · Free view-level / Paid comparison-level
  • Connect G2 → HubSpot via Native Integration
    In G2 admin → Integrations → HubSpot → Connect → Authorize. Configure: when a company views your G2 profile → update HubSpot company record with "G2 Profile View" tag + set Intent Source = G2 + add 20 points to Intent Score field.
    G2 → HubSpot Mapping
    G2 Signal:  Profile Viewed           → HubSpot: Intent Source = "G2", +20 points
G2 Signal:  Competitor Comparison     → HubSpot: Intent Source = "G2", +30 points
G2 Signal:  Review Read               → HubSpot: Intent Source = "G2", +10 points
G2 Signal:  Category Page (your cat)  → HubSpot: Intent Source = "G2", +15 points
    Native G2 ↔ HubSpot integration · No code needed
4

Connect Apollo.io → HubSpot + Google Sheet

Contact-level intent data and decision-maker enrichment — free to start

Week 2 · Day 2 ✓ Free tier
  • Create Apollo.io Account & Set ICP Filters
    Go to apollo.io → Sign up free → Go to Search → Set Company Filters: Industry (FMCG, BFSI, Pharma, Tech), HQ Location (India), Employees (500+), Keywords (marketing, brand, events). Set Contact Filters: Title contains (CMO, Marketing Head, Brand Manager, Marketing Director).
    apollo.io · 100 exports/month free
  • Connect Apollo → HubSpot (Native Sync)
    In Apollo → Settings → Integrations → HubSpot → Connect. Enable: "Push contacts to HubSpot on export" and "Sync company data". When Apollo flags a contact as intent-active, it automatically creates/updates the HubSpot record.
    Apollo ↔ HubSpot · Native integration · Free
  • Weekly Export Routine: Apollo → Google Sheet
    Every Monday: Apollo → Search → Filter by "Intent Active" → Export as CSV → Paste into APOLLO_RAW tab in your Google Sheet. The MASTER_INTENT tab will auto-recalculate scores for any company on your TAL.
    Every Monday · 10 minutes

Connect Bombora → HubSpot (Week 3–4, After Trial)

The most powerful 3rd party signal — but validate with free tools first

Week 3–4 Paid · ~$500/mo
⚠️
Do This in Week 3, Not Week 1Bombora is the most powerful source but also costs money. Validate your process with free tools first (G2, Apollo, Clutch). Once your Google Sheet scoring and HubSpot pipeline are working, then add Bombora.
  • Access Bombora via HubSpot Marketplace (Easiest Route)
    In HubSpot → Marketplace → App Marketplace → Search "Bombora" → Install. Bombora has a native HubSpot integration. You pay Bombora directly and they push surge data directly into your HubSpot company records. No manual CSV needed.
    Bombora → HubSpot: What Gets Pushed Automatically
    For each TAL company in HubSpot:
  bombora_surge_score    → 0–100 (weekly updated)
  bombora_surge_topics   → "B2B Event Management, Brand Activation"
  bombora_signal_date    → "2025-02-26"

Your HubSpot workflow then:
  IF bombora_surge_score >= 70 → Intent Score += 35
  IF bombora_surge_score >= 40 → Intent Score += 20
  TRIGGER: Update Intent Level field based on new total score
  • Configure Bombora Surge Topics for Younion
    In Bombora setup: Define your intent topics. These are the keywords Bombora will track across 5,000+ B2B publisher sites. Enter all of these:

    Primary Topics (High Priority)

    • B2B Event Management
    • Experiential Marketing
    • Brand Activation Agency
    • Corporate Event Management India

    Secondary Topics (Medium)

    • Marketing Agency India
    • Digital + On-ground Campaign
    • Brand Experience Agency
    • Event Production Company
  • Upload TAL to Bombora for Matching
    Bombora → Account-Based Intent → Upload your TAL as a CSV of company domains. Bombora will match your TAL against their dataset and only alert you when those specific companies surge — filtering out irrelevant noise.

Weekly Intent Review Process

The operating rhythm that keeps the engine running — 30 minutes every Monday

Every Week 30 min/week
  • Monday 8:30 AM — Export All 6 Sources
    Set a recurring calendar reminder every Monday. Export in this order:
    1. Bombora weekly surge report (email or download CSV from dashboard)
    2. G2 buyer intent export (G2 admin → Intent → Export)
    3. Apollo intent-active contacts (Apollo → Search → Filter intent → Export)
    4. ZoomInfo in-market signals (ZoomInfo → Intent → Export)
    5. LinkedIn Company Page analytics export (Page Admin → Analytics → Export)
    6. Clutch/IndiaMart enquiry log (check email inbox, copy into sheet)
  • Monday 9:00 AM — Paste Into Google Sheet
    Paste each export into its respective raw tab (BOMBORA_RAW, G2_RAW, etc.). The MASTER_INTENT tab recalculates automatically. Review the top 10 accounts sorted by score descending.
  • Monday 9:15 AM — SDR Briefing
    Share the Looker Studio dashboard link in the team WhatsApp/Slack. The Zapier Daily Digest should have already posted in #intent-hotlist. SDR reviews High Intent list and begins calling from top to bottom.

Monthly Scoring Model Calibration

Adjust point values based on which signals actually converted to meetings

Monthly 1 hour/month
📊
Why This MattersAfter 4 weeks of data, you'll know: "Every Bombora High Surge account that sales called — 40% booked a meeting. But every LinkedIn-only account — only 8% booked a meeting." This means Bombora signals should score higher than LinkedIn signals. Adjust accordingly monthly.
  • Pull Conversion Data from HubSpot
    HubSpot → Reports → Create report: Companies where Intent Level = High AND Meeting Booked = Yes. Group by Intent Source. This shows which source produces the highest-quality signals. Recalibrate point values in your Google Sheet formulas accordingly.
